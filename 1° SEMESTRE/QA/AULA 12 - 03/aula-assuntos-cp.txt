- confiabilidade => é imune a falhas? :

  - define a capacidadede um soft manter suas capacidades operacionais, 
    dentro do desempenho estabelecido

    - Maturidade:
      - o quanto o soft se encontra livre de defeitos

    - Tolerância a falhas:
      - capacidade de manter um nível operacional adequado, 
        mesmo quando falhas ocorram

    - Recuperabilidade:
      - capacidade de manter a integridade dos dados e 
        reestabelecer seu nível operacional no caso de falha

- usabilidade => é facil de usar? :

  - efine o esforço que deve ser despendido pelo usuário
    no processo de aprendizagem e operação do soft

      - facilidade de compreender as funcionalidades
        do sftw e se este atende às necessidades do usuário

    - operacionalidade: 
      - especifica o esforço q deve ser dispendido 
        pelo usuário para operar o soft
